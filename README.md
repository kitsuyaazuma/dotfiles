# dotfiles

This is [kitsuyaazuma](https://github.com/kitsuyaazuma)'s dotfiles.

## Installation

```bash
git clone git@github.com:kitsuyaazuma/dotfiles.git
cd dotfiles
```

### Vim

```bash
chmod +x ./setup-vim.sh
./setup-vim.sh
```

### Neovim

```bash
chmod +x ./setup-nvim.sh
./setup-nvim.sh
source ~/.bashrc
# source ~/.zshrc
```
